  • .addClass() : 선택한 요소의 클래스를 추가합니다.
  • .attr() : 선택한 요소의 HTML 속성 값을 설정하거나 반환합니다.
  • .offset() : 선택한 요소의 위치 좌표 값(문서 기준)을 설정하거나 반환합니다.
  • .scrollTop() : 선택한 요소의 세로 스크롤 값을 설정하거나 반환합니다.
  • .toggleClass() : 선택한 요소에 클래스의 추가/제거를 연속하여 사용할 수 있습니다.
